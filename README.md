# Warplane 1945 🛩️

Warplane 1945 is a high-octane 2D top-down arcade shooter (Bullet Hell) built with React and HTML5 Canvas. Experience the thrill of classic arcade combat with modern web technologies.

## 🚀 Live Demo
**[Play Warplane 1945 on Vercel](https://warplane-jet.vercel.app)**

## ✨ Key Features
- **Arcade Feel:** Smooth 60FPS gameplay with classic "Press Start" screen.
- **Dynamic Combat:** 
  - **Auto-attack:** Your plane fires automatically.
  - **Special Weapon (Bomb):** Clear all enemy bullets and deal massive damage with a screen-clearing explosion.
- **Power-up System:**
  - **(P) Power Up:** Upgrade your firepower (up to 3 levels).
  - **(H) Health:** Recover HP during intense battles.
  - **(B) Bomb:** Replenish your special weapon count.
- **Challenging Boss:** Reach 5,000 points to face the final boss with complex bullet patterns.
- **Visual Effects:** Screen shaking, explosions, particles, and floating text for an immersive experience.

## 🎮 Controls
- **Move:** `WASD` or `Arrow Keys`
- **Bomb:** `Space Bar`
- **Start Game:** `Space`, `Enter`, or `Z` (at the Start Screen)

## 🛠️ Tech Stack
- **Frontend:** React 18 (TypeScript)
- **Rendering:** HTML5 Canvas API
- **Bundler:** Vite
- **Styling:** Vanilla CSS
- **Deployment:** Vercel

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jeongjunho98/warplane.git
   ```
2. Navigate to the project folder:
   ```bash
   cd warplane
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Build for production:
   ```bash
   npm run build
   ```
