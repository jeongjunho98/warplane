# Warplane 1945 PRD (Product Requirements Document)

## 1. 프로젝트 개요
*   **프로젝트명:** Warplane 1945
*   **장르:** 2D 탑다운 스크롤 슈팅 게임 (Bullet Hell 슈팅)
*   **플랫폼:** Web (PC/Mobile 브라우저)
*   **핵심 가치:** 고전 아케이드의 손맛, 화려한 탄막, 성장의 재미

## 2. 핵심 게임 메카닉
*   **플레이어 제어:** 마우스, 키보드(WASD/방향키), 또는 터치 드래그를 이용한 8방향 자유 이동
*   **전투 시스템:**
    *   **자동 공격:** 기체는 일정 간격으로 기본 총알을 발사
    *   **필살기(폭탄):** 화면 전체 적 탄환 제거 및 강력한 데미지 (횟수 제한)
*   **아이템/파워업:**
    *   **Power Up (P):** 주무기 화력 강화 (최대 5단계)
    *   **Health (H):** 체력 회복
    *   **Bomb (B):** 필살기 횟수 추가

## 3. 적 기체 및 패턴
*   **일반 전투기:** 직선 하강하며 단발 사격
*   **중형 폭격기:** 높은 체력, 부채꼴 형태의 탄막 발사
*   **보스 기체:** 각 스테이지 끝에 등장, 다단계 페이즈와 복합 탄막 패턴

## 4. 기술 스택
*   **Frontend:** React (TypeScript)
*   **Rendering:** HTML5 Canvas API
*   **Styling:** Vanilla CSS
*   **State Management:** React Hooks

## 5. 단계별 개발 계획
1.  **Phase 1: 기반 구축** - Vite + React 세팅, Canvas 렌더링 엔진 초기화
2.  **Phase 3: 핵심 로직** - 플레이어 이동, 탄환 발사, 적군 생성 및 충돌 판정
3.  **Phase 3: 게임 요소** - 파워업 아이템, 스코어링, 생명력(HP) 시스템
4.  **Phase 4: 비주얼 & 이펙트** - 스프라이트 이미지 적용, 배경 스크롤, 폭발 애니메이션
5.  **Phase 5: 보스전 & 마무리** - 보스 패턴 구현, 게임 오버/승리 UI, 사운드 효과
